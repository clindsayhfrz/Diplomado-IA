# Diplomado-IA
Proyecto Integración ML Diplomado

# Predicción de asistencia a citas médicas

Este repositorio contiene el código y los datos necesarios para reproducir el análisis y los modelos de machine learning desarrollados para predecir la inasistencia de pacientes a sus citas médicas.
El proyecto fue desarrollado en Python utilizando Google Colab y está diseñado para ejecutarse de manera reproducible, sin requerir carga manual de archivos.

---

## Contenido del repositorio

- `codigo_Modelo_ML_Diplomado.ipynb`: Notebook principal con el preprocesamiento de datos, entrenamiento de modelos y evaluación de resultados.
- `data/`: Carpeta que contiene en enlace a Google Drive donde están los datasets utilizados en el análisis.
- `README.md`: Instrucciones generales de uso y ejecución.

---

## Requisitos

El notebook está pensado para ejecutarse en **Google Colab**, por lo que no se requiere instalación local.

---

## Ejecución del código

1. Descargar el notebook directamente de Github utilizando el siguiente enlace:

   https://github.com/clindsayhfrz/Diplomado-IA/blob/main/codigo_Modelo_ML_Diplomado.ipynb

2. Abrir el notebook en Google Colab
